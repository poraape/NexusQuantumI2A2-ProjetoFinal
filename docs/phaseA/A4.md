# A4 — Instrumentação Básica

- Os primeiros traces distribuídos encontram-se em `telemetry/traces.json`, contendo spans de upload e enfileiramento da API Gemini. Eles servirão como referência para padronizar o formato OpenTelemetry.
- As métricas base em `metrics/base.prom` definem alvos de latência, profundidade de fila e taxa de sucesso de exportações, permitindo configurar alertas proativos.
- O guia `docs/instrumentation.md` descreve como conectar os pontos de coleta à futura camada de agentes e como versionar os artefatos de observabilidade.

Esses itens atendem ao requisito mínimo de instrumentação para iniciar a migração para uma arquitetura multiagente observável desde a Fase A.
