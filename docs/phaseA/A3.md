# A3 — Testes de Caracterização do Pipeline

Os scripts em `tests/characterization/*.spec.mjs` garantem que o fluxo atual de upload, análise e exportação continue preservando:

- A integridade das estruturas de UI esperadas (`handleFileUpload`, `generateReportFromFiles`, `ExecutiveAnalysis`, `InteractiveChat`).
- A presença das rotinas críticas de exportação (`extrairDadosParaExportacao`, `gerarCsvERP`, validações fiscais).
- As integrações de chat e memória (`getChatResponse`, `searchIndex`, `createAndStoreIndex`).

Além disso, o arquivo `hash.json` registra um hash SHA-256 determinístico do fixture XML `sample-nfe.xml`, permitindo detectar qualquer alteração não intencional nos dados de entrada do pipeline.

Execute `npm test` para validar o comportamento atual antes de acionar a migração multiagente.
